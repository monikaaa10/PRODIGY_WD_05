# 🌤️ Weather Web Application

## 📌 Project Overview

The Weather Web Application is a responsive and interactive web application developed using **HTML, CSS, and JavaScript**. It allows users to search for real-time weather information by entering a city name or using their current location. The application fetches live weather data using the **OpenWeather API** and displays detailed weather information in a clean and modern user interface.

This project was developed as **Task 05** of the **Prodigy InfoTech Web Development Internship**.

---

## ✨ Features

- Search weather by city name
- Get weather using current location
- Display current temperature
- Display "Feels Like" temperature
- Display humidity
- Display wind speed
- Display atmospheric pressure
- Display weather condition with icon
- Live date and time
- Dynamic background based on weather conditions
- Fully responsive design
- Modern Glassmorphism User Interface

---
